## 📌 Project Overview
This project is a Machine Learning-based web application that predicts the likelihood of **Chronic Kidney Disease (CKD)** using patient medical data.
It helps in early detection by analyzing key health parameters and providing real-time predictions through an interactive interface.
---
## 🎯 Objectives
- Early detection of CKD  
- Apply Machine Learning on real-world dataset  
- Build an interactive web application  
- Improve decision-making using data  
---
## 🚀 Features
- ✔️ Data Cleaning & Preprocessing  
- ✔️ Handling Missing Values  
- ✔️ Imbalanced Data Handling using SMOTE  
- ✔️ Feature Engineering  
- ✔️ Model Training & Evaluation  
- ✔️ Model Saving using Pickle  
- ✔️ Interactive UI using Streamlit  
- ✔️ Real-time Prediction

## 📊 Dataset
The dataset contains medical attributes such as:
- Age  
- Blood Pressure  
- Blood Glucose  
- Hemoglobin  
- Serum Creatinine  
- Albumin  
- Sugar Levels
